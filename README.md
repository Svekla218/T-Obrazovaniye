# Т-Библиотека

Консольное приложение для управления личной библиотекой книг. Проект разработан в рамках отборочного задания для Т-Банка.

## Функционал
- **Добавление книг**: сохранение названия, автора, жанра, года издания и описания.
- **Просмотр списка**: отображение всех книг с возможностью сортировки по названию, автору или году.
- **Поиск**: поиск книг по ключевым словам в названии или имени автора.
- **Управление статусом**: возможность отметить книгу как прочитанную или добавить в "Избранное".
- **Удаление**: удаление ненужных книг из базы.
- **Сохранение данных**: все данные автоматически сохраняются в файл `library.json` и загружаются при старте.

## Требования
- Python 3.6 или выше.

2. Откройте терминал (командную строку) в папке с проектом.
3. Запустите приложение командой:
